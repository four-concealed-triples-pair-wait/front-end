# plateau-web-app

3D 都市モデル「Plateau」を使用したプログラミングと統計機械学習の学習用 Web サービス。

## 開発環境構築

以下にローカルに開発環境を構築する手順を示す。

1. 本リポジトリを git clone する。

```bash
# sshの場合
git clone git@github.com:four-concealed-triples-pair-wait/plateau-web-app.git
```

2. 必要なモジュールをインストールする。

```bash
cd app
npm install
```

3. 動作確認。開発モードで起動する。

```bash
npm run dev
```
